# Kundregister 3

Vi fortsätter med kundregistret.

Lägg till en **många-till-många-relation** för kundregistret. Det kan t.ex vara:
- En kund kan vara intresserad av en eller flera produkter
 
Skapa CRUD-operationer för detta så användaren kan administrera detta i console-appen.
